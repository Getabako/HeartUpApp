# SEO最適化チェック

対象のコンテンツ・ページに対してSEO最適化のチェックを行い、改善案を提示してください。

## チェック項目

### 1. メタデータ
- [ ] **titleタグ**: 60文字以内、主要キーワードを含む、ユニーク
- [ ] **meta description**: 120文字以内、アクションを促す内容、キーワードを含む
- [ ] **canonical URL**: 正しく設定されているか
- [ ] **OGP/Twitterカード**: 適切に設定されているか

### 2. 見出し構造
- [ ] **H1タグ**: ページに1つ、主要キーワードを含む
- [ ] **見出し階層**: H1→H2→H3の順で適切にネスト
- [ ] **キーワード配置**: 主要な見出しにキーワードを含む

### 3. コンテンツ品質
- [ ] **キーワード密度**: 1-3%程度（過剰でないか）
- [ ] **コンテンツ長**: 十分な文字数（目安: 1000文字以上）
- [ ] **読みやすさ**: 短い段落、適切な改行
- [ ] **オリジナリティ**: 独自の価値を提供しているか

### 4. 画像最適化
- [ ] **alt属性**: すべての画像に適切なalt文
- [ ] **ファイル名**: 意味のあるファイル名
- [ ] **ファイルサイズ**: 適切に圧縮されているか
- [ ] **次世代フォーマット**: WebP等の使用

### 5. 内部リンク
- [ ] **アンカーテキスト**: 説明的なテキスト
- [ ] **リンク構造**: 重要ページへの適切なリンク
- [ ] **壊れたリンク**: 404エラーがないか

### 6. テクニカルSEO
- [ ] **モバイル対応**: レスポンシブデザイン
- [ ] **ページ速度**: 読み込み時間の最適化
- [ ] **構造化データ**: Schema.orgマークアップ
- [ ] **サイトマップ**: XML sitemapの存在
